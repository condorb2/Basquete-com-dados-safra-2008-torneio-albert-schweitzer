# Basquete-com-dados-safra-2008-torneio-albert-schweitzer
Análise dos destaques do Brasil no Torneio Albert Schweitzer sob outra perspectiva: comparação de plus/minus de Mathias Alessanco, João Gluck e Pedrinho Figueiredo; impacto dos rebotes ofensivos de Isaiah Santos; relação assistências/turnovers de Pietro Melo vs rivais; e desempenho de Edu Santos contra pontuadores adversários do banco.
